# Portfolio Like Site

ポートフォリオサイトのテンプレートです。

## 使い方

1. `index.html` をブラウザで開く
2. 名前、プロフィール、制作物、スキル、メールアドレスを自分用に変更する
3. GitHub Pages / S3 + CloudFront / Netlify などで公開する

## ファイル構成

- `index.html`: ページ本体
- `style.css`: デザイン
- `script.js`: スクロール表示アニメーション、簡易メニュー

## GitHub Pages で公開する場合

CSS と JavaScript は `./style.css`、`./script.js` の相対パスで読み込んでいます。
リポジトリ名のサブパスで公開されても、そのまま表示できます。
